คีย์ลัดการทำงานระบบ (Quick Actions)

เพิ่มพนักงานใหม่ บันทึกข้อมูลบุคลากรเพิ่ม
ค้นหารายชื่อ เปิดดูข้อมูลพนักงานทุกคน
สำรองข้อมูล JSON ดาวน์โหลดพนักงานทั้งหมด
การตั้งค่าระบบ กู้คืนหรือตั้งค่าเริ่มต้น

จัดการฐานข้อมูลระบบ (System Settings)

คุณสามารถส่งออกข้อมูลพนักงานทั้งหมดเพื่อสำรองเก็บไว้เป็นไฟล์ JSON หรือทำการกู้คืนข้อมูลโดยเลือกไฟล์ที่เคยส่งออกไว้ รวมถึงการล้างข้อมูลกลับไปเป็นค่าเริ่มต้นจากทางระบบ

ส่งออกข้อมูล (Backup Data)

บันทึกข้อมูลพนักงานทั้งหมดที่มีการอัปเดตปัจจุบันลงเครื่องคอมพิวเตอร์ของคุณเป็นไฟล์ JSON

นำเข้าข้อมูล (Restore Data)

เลือกไฟล์สำรองข้อมูลพนักงาน (.json) เพื่ออัปโหลดและกู้คืนเข้าสู่ระบบนี้

รีเซ็ตข้อมูลระบบ (System Reset)

ล้างข้อมูลที่มีการบันทึกไว้ในคอมพิวเตอร์เครื่องนี้ทั้งหมด แล้วคืนค่ากลับเป็นพนักงานเริ่มต้น 47 คน
